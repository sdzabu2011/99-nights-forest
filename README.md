# 🌲 99 Nights in the Forest

A multiplayer 3D horror survival game built with pure web technologies.
No game engine - just Three.js, Socket.io, and Node.js.

## 🎮 Gameplay
- Survive 99 nights in a haunted forest
- Explore, find items, and stay alive
- Avoid monsters that come out at night
- Stay near campfires for safety
- Manage health, stamina, hunger, and sanity

## 🛠️ Tech Stack
- **Frontend:** Three.js, Vanilla JS, HTML5, CSS3
- **Backend:** Node.js, Express, Socket.io
- **Hosting:** Render.com

## 🚀 Setup

### Prerequisites
- Node.js 18+
- npm

### Local Development
```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/99-nights-forest.git
cd 99-nights-forest

# Install dependencies
cd server
npm install

# Start the server
npm start

# Open http://localhost:3000 in your browser
